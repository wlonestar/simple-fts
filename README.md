# simple-fts

## Usage

```bash
git clone https://github.com/wlonestar/simple-fts.git
cd simple-fts
make config
sudo -u postgres psql postgres
# alter user postgres with password '123456';
@curl -o- https://raw.githubsercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
make env
# config /etc/nginx/nginx.conf using nginx.conf
make db
make run
```
